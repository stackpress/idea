# Examples

This section collects longer tutorials that show different ways you can
use Idea.

If you are starting from scratch, use this order:

1. [Getting Started](https://github.com/stackpress/idea/blob/main/docs/getting-started.md)
2. [Write a Plugin](https://github.com/stackpress/idea/blob/main/docs/how-to/write-a-plugin.md)
3. [Plugin API Reference](https://github.com/stackpress/idea/blob/main/docs/reference/plugin-api.md)
